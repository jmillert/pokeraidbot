POKERAIDBOT

Support for Pokemon Go raiders using Discord
So what's this about?

This is a Discord Pokemon go raiding bot, used to announce raids, sign up for them and get an overview. Also contains commands for getting information about pokemons, raid bosses and gyms.

NOTE: In order to be able to add me to a server, you need to ensure your region's data gets imported and configured to be used on your server. For now, contact the author via the e-mail link below to get this done.

When the above step is done, to add me to your server please use this link:

Usage in chat: !raid usage

Source code and example screenshots available at Github.
